# 📍 Vacunas Quito: Buscador del Esquema Regular y Fiebre Amarilla

![Salud Pública](https://img.shields.io/badge/Sector-Salud%20P%C3%BAblica-blue)
![Ubicación](https://img.shields.io/badge/Regi%C3%B3n-Quito%2C%20Ecuador-red)
![Licencia](https://img.shields.io/badge/Licencia-MIT-green)

Esta herramienta interactiva permite a los ciudadanos del Distrito Metropolitano de Quito consultar de forma rápida y clara los servicios de inmunización en los Centros de Salud del Ministerio de Salud Pública (MSP).

## 🚀 Propósito del Proyecto
Optimizar el acceso a la información sanitaria, permitiendo filtrar los centros de salud por parroquia, horarios generales y días específicos para la vacunación contra la **Fiebre Amarilla**, facilitando la logística de los usuarios y reduciendo la carga administrativa en los puntos de atención.

## ✨ Características Principales
* 🔍 **Búsqueda Inteligente:** Localiza centros por nombre, código de establecimiento o distrito.
* Parroquias:** Filtro por zonas geográficas específicas de Quito.
* 📅 **Fiebre Amarilla:** Identificación clara del día de la semana dedicado a esta vacuna.
* 🗺️ **Integración con Google Maps:** Enlace directo para navegación GPS hacia cada centro.
* 🗓️ **Filtro de Fines de Semana:** Encuentra unidades de salud (Tipo C) con horarios extendidos.

## 🛠️ Tecnologías Utilizadas
* **React.js** para la interfaz interactiva.
* **Tailwind CSS** para un diseño responsivo y moderno.
* **Lucide React** para iconografía médica intuitiva.

## 📊 Fuente de Datos
La información contenida en esta aplicación proviene de la base de datos oficial del **Ministerio de Salud Pública del Ecuador (MSP)**. Los datos se actualizan periódicamente para reflejar cambios en la Red de Salud.
